# Un IMDBad bien moche

Backend PHP Docker, donc pour faire tourner tout ça :
````shell
docker-compose up -d &&
cd ./frontend &&
npm start
````

Puis penser à effectuer :
````shell
npm install #dans /frontend/

composer install #dans /backend/
````
